# Student Management System (CLI)

A simple Python CLI project for managing students using CRUD operations,
CSV storage, validation, and reporting.
## Features
- Add, edit, delete, search students
- List of all students
- Save and load data using CSV files
- Data validation
- Basic reports (total students, average grade)
- Command-line interface (CLI)
## Project Structure
project/
│
├── students.py        # Student entity
├── manager.py        # CRUD logic
├── csv_handler.py    # CSV load/save
├── validator.py    # Input validation
├── main.py           # CLI interface
└── README.md   
## How to Run

1. Make sure Python 3 is installed
2. Clone the repository
3. Run the program:

```bash
python main.py
```
